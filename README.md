# Data Job Radar

I was bored of the job hunt, so I built this instead.

Turns out refreshing five job boards a day, with bad filters and worse duplicates, is a great way to make a 10-minute task take 2 hours. So I wired together two job APIs, added the filters I actually wanted, and pointed it at the kind of roles I'm after.

**Live** at: https://data-job-radar.vercel.app/

## What it does

Searches [Adzuna](https://www.adzuna.co.uk/) and [Reed](https://www.reed.co.uk/) for:

- Data Analyst / Data Scientist roles
- Graduate schemes
- Internships and placements
- Entry-level data jobs

Across London, Brighton & Hove, or remote UK. Pick a role, pick a location, add keywords if you're fussy, hit search. Results update every 10 minutes so you're not staring at the same five jobs you saw yesterday.
